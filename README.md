# Dual-Tone-Multi-Frequency and sinusoidal-signal-sampling
Sinüzoidal işaretlerin birleştirilerek nasıl anlamlı bilgi (telefon tuş sesleri) oluşturduğunu kavramak, sinyal toplama prensibini uygulamak ve kullanıcı etkileşimli (GUI/Web) bir yazılım geliştirme becerisi kazanmaktır. 
Analog işaretlerin bilgisayarda ancak örneklenerek (dijital) gösterilebileceğini ve bu noktada örnekleme frekansının önemini uygulamalı olarak anlamaktır.
